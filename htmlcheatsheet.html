HTML Cheatsheet

STRUCTURE

  • div : Block container
  • span : Inline container
  • p : Paragraph(Block)
  • br : Line break
  • hr : Horizontal rule
  • h1 to h6 : Headings

LINKS

  • a href = " " : Page Link
  • a href = "mailto: " : Email link
  • a name = "name" : Anchor
  • a href = "#name" : Link to anchor

  • th : Header cell

IMAGE

FORMS

  • input type=" " : Input Field Type: text, password, checkbox, submit etc.
